# Estructura
::: tip
*Es una de las estructuras básica para poder empezar nuestros proyectos. "Los nombre de las carpetas y archivos pueden cambiar según criterio de cada uno"*
:::

```
┌── Nombre del Proyecto
|   └── frontend 
│       └── src  
│           ├── static
│           │   ├── fonts 
│           │   ├── images
│           │   ├── styles
│           │   │   ├── base
│           │   │   ├── componets
│           │   │   ├── plugins
│           │   │   ├── helpers
│           │   │   ├── layout
│           │   │   └── index.scss
│           │   └── scripts
│           │       ├── modules
│           │       ├── plugins
│           │       └── index.js
│           └── templates
│               ├── base
│               ├── section
│               └── index.pug
├── package.json
├── README.md
├──.gitignore
└── webpack.mix.js
```
